## [1:30–2:15] THE SOLUTION — Consensus Hardening Protocol

**[VISUAL]** Animated flow diagram:
- Agent recommendation → EXPLORING state
- Three agents (Finance, Strategy, Compliance) analyze independently
- Cross-model validation across multiple LLMs
- Adversarial attack by challenge agent
- PROVISIONAL_LOCK → human review → LOCKED

**NARRATION:**
"But here's where it gets interesting. That recommendation doesn't go straight to the approver — it first enters the Consensus Hardening Protocol.

In the EXPLORING state, three specialized agents — Finance, Strategy, and Compliance — independently analyze the decision from their domain perspective. Then, cross-model validation runs the analysis across multiple LLM providers. Different models might disagree — and that's actually a feature. A dedicated adversarial challenge agent then tries to find weaknesses, contradictions, or risks. Only when every validation gate passes does the decision advance to PROVISIONAL_LOCK, where the approver sees the full audit trail before LOCKING it in."

## RECORDING TIPS

1. **Screen recording:** Use OBS Studio, Loom, or built-in screen recorder
2. **Audio:** Use a decent microphone; record in a quiet room
3. **Timing:** Practice each section before recording; aim for conversational pace (~150 words/min)
4. **Dashboard:** If live demo isn't possible, use screenshot slides with cursor animations in PowerPoint/Keynote
5. **Transitions:** Use simple fade-to-black between sections (0.5s)
6. **Resolution:** 1920x1080 minimum, export as MP4
7. **Background music:** Optional — use royalty-free ambient tech music at low volume
